# پیش نیاز های پروژه

- php >= 8.1
- xampp(php8.1) >=
- postman

برید داخل پوشه ی پروژه و ی ترمینال باز کنید

```bash
composer install
```

```bash
php artisan key:generate
```

```bash
php artisan migrate
```
